## 소개

히라가나(ひらがな)와 가타카나(カタカナ)를 재미있게 학습할 수 있는 Flutter 퀴즈 앱입니다.
일본어 초보자가 빠르게 가나 문자를 익힐 수 있도록 설계했습니다.

---

## 주요 기능

- 🎌 **히라가나 퀴즈** — あいうえお 전체 문자 랜덤 출제
- 🔤 **가타카나 퀴즈** — アイウエオ 전체 문자 랜덤 출제
- ✅ **즉각 피드백** — 정답·오답 즉시 확인
- 📊 **결과 화면** — 퀴즈 종료 후 정답률 표시

---

## 기술 스택

| 분류 | 기술 |
|------|------|
| Framework | Flutter 3.x |
| Language | Dart 3.9 |
| UI | Material Design 3 |
| Platform | Android / iOS |

---

## 로컬 실행

```bash
git clone https://github.com/M1NiDRAG0N/Hika-flutter.git
cd Hika-flutter
flutter pub get
flutter run
```

> Flutter SDK 3.x 이상 필요

---

## 프로젝트 구조

```
lib/
+-- main.dart    # 앱 진입점
+-- view/        # UI 화면 (퀴즈, 결과)
+-- data/        # 히라가나 / 가타카나 데이터
```
